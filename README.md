#weixin

### 安装

clone本仓库， 在仓库根目录下 运行
```
npm install
cp config.example.coffee config.coffee
```

根据说明修改相关参数

### 功能


#### 0. 帮助

消息格式为
```
-h
或者
--help
```

#### 1. 发邮件功能

消息格式为

```
email|email-address|title|content
如
email|xxx@qq.com|测试|hello world
```
### 2.查天气

消息格式

```
-w|地点
```
